RDML-Tools

Tools to handle RDML files.

Get help · RDML-Tools · Citation · Source

RDML-Edit

RDML-Edit allows to view and edit RDML files.

RDML-LinRegPCR

RDML-LinRegPCR analyzes single runs of a RDML file.

RDML-Analyze

RDML-Analyze analyzes experiments of a RDML file.

RDML-RunView

RDML-RunView views single runs of a RDML file.

RDML-RunEdit

RDML-RunEdit edits errors and notes of a single run.

RDML-AnnotationEdit

RDML-AnnotationEdit edits the annotations of the samples.

RDML-Merge

RDML-Merge combines data from two separate RDML files.

RDML-Validate

RDML-Validate is a tool to validate RDML files against the RDML schema.

RDES-TableShaper

RDES-TableShaper reshapes a table in RDES format for the import into RDML.

RDML-BarGraph

RDML-BarGraph creates bar graphs with individual values for publication.

RDML-TestStat

RDML-TestStat calculates basic statistics on groups.

RDML-Statistics

RDML-Statistics prints statistics about RDML-Tools usage.

RDML-Tools Help

RDML-Tools Help provides an introduction into the RDML-Tools.
